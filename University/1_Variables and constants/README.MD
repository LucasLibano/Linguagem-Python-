📘 Manipulação de Variáveis e Constantes
📌 Objetivo

Este projeto tem como finalidade apresentar e praticar os conceitos fundamentais de variáveis e constantes na programação utilizando Python.

Durante as atividades, foram trabalhados:

criação de variáveis

armazenamento de dados

tipos primitivos

entrada de dados pelo usuário

conversão de tipos

uso de constantes

operações básicas

Esses conceitos são a base para qualquer linguagem de programação.

🧠 O que é uma Variável?

Uma variável é um espaço na memória do computador utilizado para armazenar informações que podem mudar ao longo da execução do programa.

Ela permite guardar dados como:

números

textos

valores lógicos

resultados de cálculos

As variáveis tornam o programa dinâmico, pois possibilitam alterar valores conforme a necessidade do usuário ou da lógica do sistema.

Exemplos de uso

guardar a idade de uma pessoa

armazenar o preço de um produto

registrar o nome de um usuário

salvar o resultado de uma conta

Sem variáveis, não seria possível criar programas interativos.

🔒 O que é uma Constante?

Uma constante é um valor que não deve ser alterado durante a execução do programa.

Ela é usada quando temos informações fixas, como:

taxas

limites

configurações

valores matemáticos

Por convenção, constantes são escritas com letras maiúsculas, facilitando sua identificação no código.

Exemplos de uso

taxa de serviço

porcentagem de desconto

valor de PI

número máximo de tentativas

O uso de constantes melhora:

organização

segurança

legibilidade

manutenção do código

Utilizados para representar números sem casas decimais, como idade, quantidade ou contadores.

🔵 Tipos numéricos decimais (float)

Utilizados para valores com casas decimais, como altura, peso, notas ou medições.

Representam verdadeiro ou falso, muito usados em condições e decisões.

🔄 Conversão de tipos

Aprendizado sobre transformar dados de um tipo para outro, como converter texto em número para realizar cálculos.

🎯 Importância desses conceitos

Variáveis e constantes são essenciais porque permitem:

armazenar informações

realizar cálculos
